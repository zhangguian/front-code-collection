# 钱包卡片堆叠

Wallet Cards

## 描述
钱包卡片堆叠展示，hover时展开所有卡片。

## 技术实现
- CSS transform堆叠
- hover展开动画
- 层次感阴影
- 过渡动画

## 使用方法
```html
<div class="wallet-container">
  <div class="wallet-card card-1">Card 1</div>
  <div class="wallet-card card-2">Card 2</div>
  <div class="wallet-card card-3">Card 3</div>
</div>
```
