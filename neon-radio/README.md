# 霓虹单选按钮

Neon Radio

## 描述
霓虹灯风格单选按钮，包含clip-path动画效果。

## 技术实现
- CSS clip-path裁剪
- 霓虹发光效果
- 动画过渡
- 选中状态样式

## 使用方法
```html
<div class="neon-radio-group">
  <label class="neon-radio">
    <input type="radio" name="neon" value="1" />
    <span class="radio-mark"></span>
    <span class="radio-text">Neon 1</span>
  </label>
  <label class="neon-radio">
    <input type="radio" name="neon" value="2" />
    <span class="radio-mark"></span>
    <span class="radio-text">Neon 2</span>
  </label>
</div>
```
