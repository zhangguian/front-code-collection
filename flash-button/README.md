# Flash Button

Button with flash/lightning effect

## 描述
悬停时黄色多次闪烁光效，文字颜色变化

## 技术实现
- 多段keyframe动画实现闪烁
- radial-gradient背景光效
- filter模糊创建光晕

## 使用方法
```html
<button>
    <div class="svg-1">
        <svg viewBox="0 0 58.56 116.18">
            <path d="M51.68 79.32c-5.6,0.48..." class="fil0"></path>
        </svg>
    </div>
    <p>Button</p>
</button>
```
