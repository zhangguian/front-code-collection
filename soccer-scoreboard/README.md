# 足球比分板

Soccer Scoreboard

## 描述
足球比分板组件，采用斜切设计风格。

## 技术实现
- CSS skew斜切效果
- 比分数据展示
- 动态更新
- 响应式布局

## 使用方法
```html
<div class="scoreboard">
  <div class="team home">
    <span class="team-name">Home</span>
    <span class="score">2</span>
  </div>
  <div class="divider">:</div>
  <div class="team away">
    <span class="score">1</span>
    <span class="team-name">Away</span>
  </div>
</div>
```
