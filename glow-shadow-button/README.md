# Glow Shadow Button

Glassmorphism button with glowing shadow

## 描述
毛玻璃按钮，hover时蓝色霓虹光从上下扩散并充满

## 技术实现
- backdrop-filter 毛玻璃效果
- ::before/::after 上下光效扩散
- box-shadow 霓虹发光
- skewX 光泽扫过效果
- letter-spacing 文字展开

## 使用方法
```html
<div class="container">
    <div class="btn"><a href="#">Read more</a></div>
</div>
```
