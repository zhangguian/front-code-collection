# 新拟态开关

Neo Toggle

## 描述
新拟态风格开关，包含频谱分析仪动画效果。

## 技术实现
- CSS新拟态阴影
- 频谱动画效果
- 过渡动画
- 状态切换

## 使用方法
```html
<label class="neo-toggle">
  <input type="checkbox" class="toggle-input" />
  <span class="toggle-track">
    <span class="toggle-thumb"></span>
    <span class="spectrum"></span>
  </span>
</label>
```
