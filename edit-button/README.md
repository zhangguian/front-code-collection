# Edit Button

Edit button with hover animation

## 描述
悬停时模糊圆形扩散，编辑图标旋转移动，底部出现线条

## 技术实现
- CSS伪元素模糊扩散效果
- transform旋转+位移动画
- scaleX线条绘制动画

## 使用方法
```html
<button class="editBtn">
    <svg height="1em" viewBox="0 0 512 512">
        <path d="M410.3 231l11.3-11.3..."></path>
    </svg>
</button>
```
