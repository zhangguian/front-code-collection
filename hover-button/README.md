# Hover Button

Gradient hover button with icon

## 描述
圆形渐变按钮，悬停放大+发光阴影效果

## 技术实现
- linear-gradient渐变背景
- transform缩放动画
- box-shadow发光效果

## 使用方法
```html
<button class="hover-button">
    <svg viewBox="0 0 24 24" xmlns="http://www.w3.org/2000/svg">
        <path d="M12 2a10 10 0 100 20 10 10 0 000-20zm-1 15v-4H8l4-6v4h3l-4 6z"></path>
    </svg>
</button>
```
