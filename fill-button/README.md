# 填充按钮

Fill Button

## 描述
填充按钮，从4个方向触发填充动画效果。

## 技术实现
- CSS伪元素实现填充层
- CSS animation控制填充方向
- 多方向动画效果
- hover状态触发

## 使用方法
```html
<button class="fill-btn" data-direction="left">Hover Me</button>
<button class="fill-btn" data-direction="right">Hover Me</button>
<button class="fill-btn" data-direction="top">Hover Me</button>
<button class="fill-btn" data-direction="bottom">Hover Me</button>
```
