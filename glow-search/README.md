# 发光搜索输入框

Glow Search

## 描述
发光搜索输入框，旋转渐变边框效果。

## 技术实现
- CSS conic-gradient旋转渐变
- @property动画自定义属性
- box-shadow发光效果
- 过渡动画

## 使用方法
```html
<div class="glow-search-container">
  <input type="text" class="glow-search-input" placeholder="Search..." />
  <span class="search-icon">🔍</span>
</div>
```
