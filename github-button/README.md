# GitHub Button

GitHub star button with animation

## 描述
黑色圆角按钮，悬停时星星旋转变黄+发光效果

## 技术实现
- transform rotate旋转动画
- filter drop-shadow多层发光
- scale缩放效果

## 使用方法
```html
<button class="button">
    <span class="text">Star on GitHub</span>
    <svg class="icon" viewBox="0 0 47.94 47.94" fill="currentColor">
        <path d="M26.285,2.486l5.407,10.956..."></path>
    </svg>
</button>
```
